# Inlämning 2

## Vad gör skriptet?

Det här skriptet hämtar ett slumpmässigt kattfakta från ett öppet API.

Get-RandomThing.psm1 innehåller en funktion som hämtar data från API:t. I run.ps1 anropas funktionen och resultatet skrivs ut i PowerShell.
